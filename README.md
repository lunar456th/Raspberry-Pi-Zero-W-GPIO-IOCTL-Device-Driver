# Raspberry-Pi-Zero-W-GPIO-IOCTL-Device-Driver
Linux device driver for raspberry pi zero w gpio using ioctl.

## Components
* Device Driver (mydev\_gpio)
* Shared Library (libmydev\_gpio)
* App for Test (test\_mydev\_gpio)

## Functions
* Device Driver
  * init
  * open
  * ioctl
  * release
  * exit
* Library
  * set direction
  * set value
  * get value

## Call Structure
* App -> shared library -> device driver

## How to build

1. How to build device driver
> make

2. How to build shared library
> make lib

3. How to build test app
> make test

4. How to build test app for library
> make testlib

## How to clean

1. How to clean device driver
> make clean

2. How to clean shared library
> make libclean

3. How to clean test app
> make testclean

4. Hot wo clean test app for library
> make testlibclean
